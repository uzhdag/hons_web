      Information

      The satellite workshop will be held in conjuction with NetSci 2015, the premiere international conference on network science held in Zaragoza, Spain.

      A number of practical information regarding traveling, accomodation and registration can be found already on the website of the hosting event NetSci 2015.

      Registration

      Information on how to register for both the main conference as well as the satellite can be found here. A single day registration (75 - 125 EUR) is available as well as special rates for students, postdocs and academics. Please note that there is a discount for early registrations made before April 7, 2015.

      Please note that there is a limited number of seats available for this satellite event. In addition to the registration with NetSci, we thus kindly ask you to register for this event directly with us, thus allowing us to plan the available space in advance. This registration only serves organizational purposes and is thus free.
